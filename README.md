## Running App Locally

Change into root directory to install all dependencies and start the app.

```js
npm install 

npm start
```

## Testing 

Cypress was used for testing.  To run the tests change into the **cypress-tests** folder and run the following commands: 

```js
npm install

npx cypress open
```
